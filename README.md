# House-sitting guide

A mobile-friendly home care guide for our house sitter, published with GitHub Pages.

## How it's published
The site is a single static file, `index.html`, served from the repo root.

To enable GitHub Pages: **Settings → Pages → Build and deployment → Source: Deploy from a branch → Branch: `main` / root → Save.** The site goes live at `https://<username>.github.io/housesitting/` within a minute or two.

## Privacy — please read
This published version **intentionally leaves out** the door code, Wi-Fi password, and personal phone numbers, because a GitHub Pages site is **public**. Those details are shared with the sitter directly.

Do **not** commit the full-detail print PDF (the one containing the code and numbers) to this public repo — keep it private and hand it to the sitter separately.
